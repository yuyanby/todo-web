# Todo Web - 智能任务管理系统

🔗 **在线访问**: https://yuyanby.github.io/todo-web/

## 功能特点

- 📋 **任务管理** - 添加、完成、删除任务
- 🏷️ **智能分类** - 婚礼/新房分类，优先级标记
- 🤖 **AI执行** - 支持 Hermes AI 执行任务
- 📖 **攻略系统** - 每个任务可生成完整攻略
- 🖼️ **图片支持** - 攻略包含图片画廊
- 📱 **响应式设计** - 支持手机/平板/电脑
- 🔍 **搜索筛选** - 快速找到任务
- 📊 **统计面板** - 实时查看任务进度

## 使用方法

### 1. 查看任务列表
访问 https://yuyanby.github.io/todo-web/ 查看所有任务

### 2. 筛选任务
- 点击侧边栏筛选：全部/待完成/已完成/婚礼/新房/AI执行/我执行
- 使用顶部搜索框搜索任务
- 使用优先级标签筛选

### 3. 查看攻略
点击任务卡片上的"📖 查看攻略"按钮，查看详细攻略

### 4. 标记完成
点击任务卡片上的"✓ 完成"按钮标记任务完成

### 5. 添加任务
点击右上角"+ 添加任务"按钮添加新任务

## 技术栈

- HTML5
- CSS3 (Flexbox + Grid)
- Vanilla JavaScript
- Font Awesome 图标
- GitHub Pages 托管

## 本地开发

```bash
# 克隆仓库
git clone https://github.com/yuyanby/todo-web.git
cd todo-web

# 启动本地服务器
python3 -m http.server 8080

# 浏览器访问 http://localhost:8080
```

## 数据结构

任务数据存储在 `js/data.js` 中，包含以下字段：

```javascript
{
    id: 1,                    // 任务ID
    content: "任务内容",       // 任务描述
    status: "pending",        // 状态: pending/completed
    priority: "high",         // 优先级: high/medium/low
    category: "新房",         // 分类: 婚礼/新房
    owner: "hermes",          // 负责人: hermes/user
    due_date: "2025-05-25",   // 截止日期
    output_file: "tasks/task_1.html",  // 攻略文件路径
    created_at: "2025-05-19" // 创建时间
}
```
